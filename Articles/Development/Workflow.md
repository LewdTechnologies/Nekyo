
# Development Workflow

*How to use your local environment.*

<br>

## General Steps

<br>

1.  Setup your local environment.

    <br>
    
2.  Write / modify code.

    <br>
    
3.  Execute the run script.

    ```sh
    Tools/.js run
    ```
    
    *All scripts are run from the project directory.*

    <br>

4.  Repeat steps 2 - 3.

<br>
